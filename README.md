# harayadev

Monorepo con dos partes:

- **`starter-kit/`** — el producto: template Next.js config-driven para generar
  webs con chat IA para pymes chilenas. Ver `starter-kit/README.md` para comandos,
  arquitectura y flujo por cliente.
- **raíz** — (pendiente de construir) el sitio que vende este starter kit: landing
  con las demos/presets, casos, precios, etc. También se despliega en Vercel, como
  proyecto Vercel separado apuntando a la raíz del repo (mientras que cada cliente
  usa un proyecto Vercel con Root Directory = `starter-kit`).
